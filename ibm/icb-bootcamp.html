Abstract

This intensive 5-day bootcamp provides comprehensive training on IBM App Connect, IBM Cloud Pak for Integration, Red Hat OpenShift, and Event Streams. Designed for developers and administrators, this course covers installation, configuration, management, and practical deployment scenarios, enabling participants to leverage these technologies effectively in real-world environments.

Objectives
  • Install, configure, and manage IBM App Connect and IBM Cloud Pak for Integration.
  • Deploy, manage, and scale applications using Red Hat OpenShift.
  • Develop event-driven applications using IBM Event Streams.
  • Integrate various IBM and Red Hat technologies for robust, scalable, and secure enterprise solutions.
  • Implement CI/CD pipelines and GitOps practices for continuous integration and deployment.
Topics
  1. IBM App Connect
    • Overview and architecture
    • Installation and configuration
    • Message flow development and deployment
    • Integration with IBM MQ
    • Security and error handling
    • Performance tuning
  2. IBM Cloud Pak for Integration
    • Introduction and architecture
    • Installation and configuration
    • Integration capabilities
    • API management and App Connect
    • Event Streams and DataPower
  3. Red Hat OpenShift
    • Kubernetes fundamentals
    • OpenShift setup and management
    • Application deployment and scaling
    • CI/CD with OpenShift Pipelines
    • Security and compliance
  4. Event Streams Developer Essentials
    • Introduction to Apache Kafka and IBM Event Streams
    • Setting up Kafka clusters
    • Producing and consuming messages
    • Real-time data processing
  5. OpenShift for Developers
    • Developer tools and workflows
    • Building and deploying applications
    • Using OpenShift Dev Spaces and Developer Hub
    • Integrating with CI/CD pipelines
