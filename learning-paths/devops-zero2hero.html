DevOps Zero2Hero by Skunkworks - Introduction Chapter

Date: 23 July 2024
Author: Raydo Matthee

Overview

This guide is designed to provide structured exercises and learning paths for Haripriya to enhance her understanding and skills in the DevOps environment she will be supporting at Lloyds Bank. The course is self-paced and covers essential platforms and tools for modern DevOps practices.

1. Account Setup

Objective:

Set up essential accounts to access required platforms and tools.

  1. HashiCorp Cloud Platform (HCP)

    Sign up at: HashiCorp Cloud Platform

  2. Harness Platform

    Sign up or sign in at: Harness Platform

  3. GitHub

    Sign up at: GitHub

Self-Assessment Criteria:

  • Successfully create accounts and verify email addresses.
  • Ensure you can log in to each platform without issues.

Knowledge Check:


  1. Have you successfully created accounts on HashiCorp Cloud Platform, Harness Platform, and GitHub?
  2. Can you log in to each platform without encountering any issues?
  3. Have you verified your email addresses for all the accounts?

2. GitHub Learning Path


Objective:

Gain proficiency in using GitHub for version control, collaboration, and site creation.

  1. First Day on GitHub


    Course link: First Day on GitHub

  2. Communicate Using Markdown


    Course link: Communicate Using Markdown

  3. GitHub Pages


    Course link: GitHub Pages

Self-Assessment Criteria:


  • Complete each course and earn the corresponding badges.
  • Create a personal repository, use Markdown for documentation, and publish a site using GitHub Pages.

Knowledge Check:


  1. What is the purpose of using GitHub for version control?
  2. How do you create a new repository on GitHub?
  3. Can you explain the basics of Markdown and how it is used on GitHub?
  4. Have you created a personal repository and used Markdown for documentation?
  5. Can you describe the steps to publish a site using GitHub Pages?

3. Harness Platform Fundamentals


Objective:

Understand the fundamental concepts and components of the Harness Platform.

Self-Paced Training - Platform Fundamentals


Register at: Harness University

Course Outline:


  • Account, Organization, Project, Secrets: Learn the basics of account setup, organizational structure, and secure project management.
  • Platform - Delegates: Understand the role and setup of Delegates in workflow execution.
  • Platform - Connectors: Explore how to connect Harness to your SDLC systems.
  • Platform - Pipeline > Stage > Step: Dive into pipeline creation and management.
  • Platform - Variables > Expressions: Use variables and expressions to optimize pipeline processes.
  • Platform - Approvals: Implement approvals for controlled pipeline execution.
  • Continuous Integration: Learn techniques for efficient code building.
  • Continuous Delivery: Master the processes for smooth and reliable software releases.

Self-Assessment Criteria:


  • Complete the video and documentation components of each section.
  • Demonstrate the ability to set up and manage pipelines, delegates, and connectors within the Harness platform.

Knowledge Check:


  1. What are the foundational elements of the Harness platform?
  2. Can you describe the role of Delegates in the Harness platform?
  3. How do Connectors facilitate integration with SDLC systems in Harness?
  4. What are the key components of a pipeline in Harness, and how do they interact?
  5. How can Variables and Expressions be used to optimize pipeline processes in Harness?
  6. What is the importance of Approvals in pipeline execution?
  7. Describe the techniques for efficient code building in Continuous Integration.
  8. What are the best practices for smooth and reliable software releases in Continuous Delivery?

4. HashiCorp Vault Tutorials


Objective:

Learn how to secure, store, and manage secrets using HashiCorp Vault.

  1. Vault CLI


    Tutorial link: Vault CLI

  2. HCP Vault Dedicated


    Tutorial link: HCP Vault Dedicated

  3. Vault UI


    Tutorial link: Vault UI

  4. HCP Vault Secrets Quickstart


    Tutorial link: HCP Vault Secrets Quickstart

Self-Assessment Criteria:


  • Complete each tutorial and understand the fundamental operations of Vault CLI and UI.
  • Set up a managed Vault cluster using HCP.
  • Demonstrate the ability to manage secrets and configure the Vault environment.

Knowledge Check:


  1. What are the fundamental operations of the Vault CLI?
  2. How do you set up a managed Vault cluster using HCP?
  3. Can you explain how to manage secrets using the Vault UI?
  4. What are the key features of HCP Vault Secrets?
  5. Have you completed the tutorials and understand the basic operations of Vault CLI and UI?
  6. Are you able to manage and configure the Vault environment effectively?

Summary


By following this structured guide and completing the exercises, Haripriya will build a strong foundation in using essential DevOps tools and platforms, enabling her to effectively support the environment at Lloyds Bank. Each section includes specific self-assessment criteria to ensure thorough understanding and practical skills development.

Note: If any issues or questions arise during the exercises, please reach out for assistance or consult the relevant documentation and community forums for support.
